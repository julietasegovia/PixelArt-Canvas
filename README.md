# PixelArt-Canvas
A simple pixel-drawing page based on a 20x20 css grid. Each cell from the grid is clickable and represents a pixel from the drawing. 
The restart button sets every pixel to white again, the erase button enables "erasing mode", every pixel clicked during "erasing mode" will be set back to while. The erase button HAS TO BE CLICKED AGAIN to disable "erasor mode"
The script is written so you can drag the cursor around the canvas while holding click to both draw and erase.
The color-picker will immidiatly change the current color to the one selected.
